**NFC 标签读写**

当您通过 NFC 标签写入或读取功能创建 Home Assistant 自动化触发时，为了启用基于 NFC 标签的自动化，我们会通过 Android NFC API 与物理 NFC 标签通信。写入标签时，应用生成 UUID 并写入标签。读取标签时，应用扫描 NFC 标签或 QR 码，提取标签 ID，经您确认后发送至 Home Assistant 服务器触发自动化。

处理方式说明：
上传云端处理：标签 ID 通过 REST API（HTTPS加密）发送至您自托管的 Home Assistant 服务器。已授权的标签 ID 本地存储在应用偏好设置中。

2. 设备权限调用

NFC（近场通信）权限：当读取或写入 NFC 标签时，本应用将使用设备 NFC 硬件与标签通信。
拒绝后果：如果 NFC 在系统设置中关闭，NFC 标签读写功能将不可用。
