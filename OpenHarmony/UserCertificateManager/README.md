## 功能模块

### 1. 证书列表页

| 功能 | 数据流 | 隐私风险 |
|------|--------|----------|
| 查看证书 | 本地存储->证书列表->显示 | 低（本地处理） |
| 查看证书 | 系统API->设备信息->UI布局适配 | 低（本地处理） |
| 删除全部证书 | 本地存储->删除操作->系统存储 | 中（数据不可恢复） |

### 2. 证书详情页

| 功能 | 数据流 | 隐私风险 |
|------|--------|----------|
| 证书详情 | 本地存储->证书详情->显示 | 低（本地处理） |
| 删除证书 | 本地存储->删除操作->系统存储 | 中（数据不可恢复） |

### 3. 导入证书页

| 功能 | 数据流 | 隐私风险 |
|------|--------|----------|
| 导入CA证书 | 用户文件->证书读取->身份认证->安装到系统存储 | 中（需文件读取权限） |
| 导入用户凭证 | 用户文件->凭证读取->密码验证->安装到系统存储 | 中（需文件读取权限） |
| 导入系统凭证 | 用户文件->凭证读取->密码验证->安装到系统存储 | 中（需文件读取权限） |

### 4. 凭证详情页

| 功能 | 数据流 | 隐私风险 |

## 隐私数据类型

1. **证书信息**: 证书别名、颁发者、使用者、有效期、SHA256指纹
2. **凭证信息**: 凭证别名、密钥数量、证书数量
3. **用户身份信息**: 指纹/PIN认证结果
4. **应用信息**: 应用UID、应用名称、应用图标
5. **密码信息**: 凭证文件密码（内存临时存储）
6. **设备信息**: 设备类型（用于UI布局适配）

## 权限要求

1. **证书管理权限**: 访问和管理本地证书存储（预授权）
2. **文件读取权限**: 从存储设备读取证书文件（用户授权）
3. **身份认证权限**: 验证用户身份（系统授权）
4. **屏幕截图保护权限**: 防止密码被截图窃取（预授权）

## 安全措施

1. 所有数据处理均在本地完成，不涉及网络传输
2. 证书存储使用系统安全存储
3. 密码在内存中临时存储，使用后立即清除
4. 密码输入页面启用防截图保护
5. 敏感操作（安装、删除）需用户确认
