设备传感器

1 我们如何收集和使用您的个人信息：
设备传感器（智能家居数据联动），为了将您的设备状态信息同步至 Home Assistant 服务器以实现自动化场景，应用会定期采集以下设备传感器数据：

（1）**运动活动数据**（敏感个人信息）：通过 CMMotionActivityManager 采集您的当前运动类型（静止/步行/跑步/驾驶等）及置信度。此数据反映您的实时活动状态。
（2）**计步数据**（敏感个人信息）：通过 CMPedometer 采集您的步数、步行距离、爬楼楼层数、步速和步频。这些数据反映您的日常运动量。
（3）**气压数据**：通过 CMAltimeter 采集设备周围的大气压和相对海拔变化。
（4）**电池数据**：通过 UIDevice 采集电池百分比和充电状态。此数据反映设备电源状况。
（5）**网络信息**（敏感个人信息）：通过 NWPathMonitor 采集当前连接的 Wi-Fi **SSID 和 BSSID**、网络连接类型（WiFi/蜂窝/以太网）；通过 CoreTelephony 采集**蜂窝运营商名称、移动国家码（MCC）、移动网络码（MNC）** 和当前无线接入技术（如 5G NR、LTE）。SSID/BSSID 可用于设备定位和唯一标识，属于敏感个人信息。
（6）**显示信息**：通过系统 API 采集连接的显示器数量和名称。
（7）**专注模式状态**：通过系统 FocusStatus API 采集设备当前的专注模式开关状态。此数据需获得您的明确授权。

采集频率：定期采集（周期性和传感器值变化时触发更新）。处理方式说明：上传至您的自托管服务器——所有传感器数据通过 HTTPS 加密传输至您自己的 Home Assistant 服务器，不传输至任何第三方服务器。您可以在"设置 -> 传感器"页面中逐个开关每个传感器。关于敏感数据的安全措施：运动数据、网络标识数据等在传输过程中全程使用 HTTPS 加密。

2 设备权限调用：
（1）运动与健身权限（NSMotionUsageDescription）：当您启用活动识别或计步器传感器时，本应用将询问运动与健身权限，以便访问您的运动活动和计步数据。如您拒绝授权，活动传感器和计步传感器不可用。
（2）专注模式权限（NSFocusStatusUsageDescription）：当您启用专注模式传感器时，本应用将询问专注模式权限，以便读取您设备的专注模式状态。如您拒绝授权，专注模式传感器不可用。
