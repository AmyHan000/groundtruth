用户身份认证

1 我们如何收集和使用您的个人信息：
用户身份认证（登录认证），当您登录 Home Assistant 服务器时，为了验证您的身份并建立安全的 API 通信，我们通过 OAuth2 协议从您的服务器获取**访问令牌（Bearer Token）和刷新令牌（Refresh Token）**。这些令牌是您服务器身份凭证的代理，属于**敏感个人信息**。所有令牌使用 iOS Keychain（硬件加密）安全存储，仅在应用与您的服务器通信时用于 API 请求头，不会传输至任何第三方。采集频率：一次性获取，令牌自动刷新。处理方式说明：本地加密存储。

设备注册，当您完成登录后，为了向您的服务器注册此设备以便接收推送通知，我们会将**设备名称、推送通知令牌、应用版本号**上传至您的 Home Assistant 服务器。其中推送通知令牌为设备唯一标识，属于**敏感个人信息**。采集频率：首次注册时触发。处理方式说明：上传至您的自托管服务器。

2 设备权限调用：
不涉及危险权限调用。
