1 我们如何收集和使用您的个人信息

【系统文件访问】用户在 iOS“文件”应用或其他支持 File Provider 的应用打开 Nextcloud 文件时。为了按需提供云端文件内容，Nextcloud iOS会处理文件标识、路径、文件内容及下载状态，数据来源为File Provider 项目标识和本地元数据，处理频率为每次系统请求尚未在本地的文件时。处理方式：扩展解析文件标识并启动后台 HTTPS 下载；存储方式：文件保存在 File Provider 管理的本地存储，元数据保存在数据库；数据接收方：iOS Files/File Provider、用户设备和 Nextcloud 服务器；第三方共享情况：不向无关第三方共享；打开文件的目标应用可读取用户选择的文件。您可通过iOS 文件 > Nextcloud；可移除下载或注销账号控制或停止该处理。

2 设备权限调用

File Provider 系统扩展能力：在用户在系统文件界面访问 Nextcloud时以由用户选择文件触发方式使用，用于按需提供云端文件。若拒绝或不可用，系统文件界面无法打开该云端文件。
